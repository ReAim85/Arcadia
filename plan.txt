We're building AgentHub—a comprehensive AI Agents Marketplace platform that transforms how developers discover, evaluate, and deploy autonomous software engineering agents. Unlike static AI tool directories, our platform enables one-click deployment of fully functional agents that execute real software engineering tasks, complete with live demos, performance metrics, trust scores, and seamless integration with existing development workflows.
Our four-person team brings complementary expertise: You serve as Team Lead and Full Stack Engineer, architecting the system integration, managing the ngrok hosting infrastructure for the live demo, and delivering the final pitch; Sagar leads Frontend Development using React, Vite, and Chakra UI to create a polished, responsive marketplace interface with rich agent showcase pages, deployment modals, and real-time debug consoles; Jeet and Krishna handle all Python backend development through FastAPI, with Jeet specifically building the Live Debugger agent and WebSocket infrastructure for real-time log streaming, while Krishna develops the API Builder agent and Vercel deployment automation—both leveraging CrewAI for agent orchestration and SQLAlchemy for database management.
The platform showcases four distinct agents: two fully operational and two simulated for demonstration breadth. The API Builder agent takes database schemas (SQL or NoSQL) as input, automatically generates complete REST APIs with authentication, rate limiting, and OpenAPI documentation, then deploys to live cloud infrastructure—returning a working URL within 90 seconds. The Live Debugger agent connects to running applications via WebSocket, captures runtime exceptions with full context including stack traces and variable states, reproduces errors in headless browsers, generates tested fixes, and automatically creates GitHub pull requests with detailed diffs. The System Architect and Security Auditor agents are presented as polished UI simulations, demonstrating roadmap capabilities for architecture visualization and vulnerability scanning interfaces.
For the HackIndia 24-hour hackathon, we're executing an intensive sprint: Hours 0-6 establish the React + Vite + Chakra UI frontend scaffold, FastAPI backend skeleton, and database schema; Hours 6-12 build the API Builder agent core with schema parsing and Vercel deployment integration; Hours 12-18 implement the Live Debugger agent with error capture, fix generation, and GitHub PR creation; Hours 18-22 integrate frontend and backend, build the 2 simulated agent UIs, and set up ngrok + VS Code port forwarding; Hours 22-24 conduct end-to-end testing, fix critical bugs, and rehearse the demo flow. You'll host the frontend via ngrok from your laptop with VS Code port forwarding exposing the FastAPI backend, enabling judges to interact with deployed APIs on their own devices. The 5-minute demo flow progresses from marketplace browsing to live API generation to autonomous bug detection and resolution—conclusively proving that AI agents can transcend conversational interfaces to deliver measurable software engineering outcomes.
